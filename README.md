Each folder is related to a challenge and is named after the challenge name and the covered topic.
Each folder contains a given_materials folder that contains the material for executing the challenge.
The folder Other_scripts contain scripts related to other topics.

## Repository Structure

- **Challenge Folders**  
Each folder challenge_name__covered_topic contains the material produced for solving a challenge.

- **given_materials/**  
  Inside every challenge folder there is also a `given_materials` directory containing the materials required to execute the challenge.

- **Other_scripts/**  
  The `Other_scripts` folder contains additional scripts related to topics outside the main challenges.
